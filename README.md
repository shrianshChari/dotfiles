# dotfiles #

<hr>

### Prerequisites:
- You are installing on Linux (presumably Debian/Ubuntu-based distros)
- You already have installed `git`.

### Installation:
<code>git clone https://<span></span>github.com/shrianshChari/dotfiles.git ~/.dotfiles</code>
